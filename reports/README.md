# Radiology AI Reports

This folder contains example radiology reports generated by the full DIP + MedSAM + LangChain + Gemini workflow.

Each report typically includes:
- Patient ID
- Clinical information (Age, Subtype, Stage Code)
- Imaging findings
- Literature synthesis based on ~20 PubMed papers
- Suggested next clinical steps
- Uncertainty / limitations

All reports are generated automatically by running:

```bash
python agent/step7_run_agent.py
